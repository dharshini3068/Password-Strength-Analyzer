# 🔐 Password Strength Analyzer

# 📌 Description
A Python-based tool that evaluates password strength based on length, complexity, and character usage. It also generates strong password suggestions to improve security and help users create safer passwords.

---

# ✨ Features
- Checks password length  
- Validates uppercase and lowercase letters  
- Checks numbers and special characters  
- Classifies password strength (Weak / Medium / Strong)  
- Generates strong random passwords  

---

# 🛠️ Technologies Used
- Python  
- Regex (re module)  
- Random module  
- String module  

---

# 🚀 How to Run

## 1. Install requirements
pip install re

## 2. Run program
python analyzer.py

## 3. Enter password when prompted
Example:
test123

---

# 📊 Sample Output
Enter password: test123

⚠ Add uppercase letters  
⚠ Add special symbols  

🔴 Weak Password  

Suggested Password: A9#kL2@pXz

---

# 🧠 How It Works
- User inputs password  
- System checks rules (length, symbols, numbers)  
- Calculates strength score  
- Generates strong password suggestion
